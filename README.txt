🚀 التثبيت (5 خطوات):
════════════════════════════════════════════════════════════════

1. رفع الملفات:
   ──────────────────────────────────────
   ارفع جميع الملفات إلى السيرفر

2. إنشاء قاعدة البيانات:
   ──────────────────────────────────────
   phpMyAdmin → أنشئ: iptv_db
   استورد: database.sql

3. تعديل config.php:
   ──────────────────────────────────────
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'iptv_db');
   define('DB_USER', 'root');
   define('DB_PASS', '');

4. تعديل client_config.php: ⭐
   ──────────────────────────────────────
   define('LICENSE_SERVER_URL', 'http://yourdomain.com/act/api.php');
   define('LICENSE_API_KEY', 'نفس_المفتاح_في_سيرفر_الرخص');

5. ضبط الصلاحيات:
   ──────────────────────────────────────
   chmod 755 /var/www/iptv
   chmod 777 /var/www/iptv/uploads

🆕 التحديث الجديد: ربط Domain
════════════════════════════════════════════════════════════════

المشكلة القديمة:
❌ نفس السيرفر → domains متعددة تعمل بدون رخصة

الحل الجديد:
✅ Machine ID = Hardware + Domain
✅ كل domain يحتاج رخصة منفصلة
✅ لا يمكن نسخ النظام

مثال:
──────────────────────────────────────
نفس السيرفر (VPS واحد):

iptv1.com:
Machine ID: abc123...def (يحتوي iptv1.com)
رخصة: IPTV-A1B2 ✅

iptv2.com:
Machine ID: xyz789...ghi (يحتوي iptv2.com) ← مختلف!
رخصة: يحتاج رخصة جديدة ✅

iptv3.com:
Machine ID: lmn456...jkl (يحتوي iptv3.com) ← مختلف!
رخصة: يحتاج رخصة جديدة ✅

⚠️ مهم للعملاء:
════════════════════════════════════════════════════════════════

عند التفعيل، يجب فتح activate.php من Domain الأساسي:

✅ صحيح: http://yourdomain.com/activate.php
❌ خاطئ: http://localhost/activate.php
❌ خاطئ: http://192.168.1.100/activate.php

لأن Machine ID يتضمن Domain!

🔄 سير العمل:
════════════════════════════════════════════════════════════════

1. العميل يركب النظام على domain.com
2. يعدّل client_config.php بمعلومات سيرفر الرخص
3. يفتح http://domain.com/activate.php
4. Machine ID يظهر (يحتوي domain.com)
5. يرسل لك Machine ID
6. تضيف رخصة في سيرفرك مع Domain
7. ترسل له License Key
8. يفعّل ✅
9. النظام يعمل

إذا حاول نسخ النظام لـ domain2.com:
→ Machine ID مختلف (يحتوي domain2.com)
→ يحتاج رخصة جديدة ✅
